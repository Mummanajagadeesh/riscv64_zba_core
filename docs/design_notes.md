## Microarchitecture Overview

- 5-stage pipeline
- No hazard forwarding (simple stalling-free test program)
- Harvard architecture
- Zba instructions implemented in execute stage

### Zba Instructions Supported
- SH1ADD
- SH2ADD
- SH3ADD

### Datapath Components
- Program Counter
- Instruction Fetch Unit
- Decode + Register File
- ALU + Zba Unit
- Data Memory Interface
- Writeback MUX
